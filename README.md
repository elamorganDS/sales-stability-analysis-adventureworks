# Sales Stability Analysis – AdventureWorks

## Project description
This is a university project for the course
**Programming Languages in Business Data Analytics**.

The goal of the project is to analyze the **stability and variability of sales**
using the **AdventureWorks Data Warehouse**.

## Scope
The project includes:
- data extraction and aggregation using SQL
- analytical and segmentation tasks
- visualization of trends, seasonality, and variability

## Technologies
- SQL (analytical queries)
- Python (data analysis and visualization)
- AdventureWorks Data Warehouse

## Structure
- `sql/` – SQL queries
- `python/` – Python scripts and notebooks
- `report/` – final report (Word / PDF)
- `data/` – auxiliary data (optional)
